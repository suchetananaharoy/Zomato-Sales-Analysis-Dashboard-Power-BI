# Zomato Sales Analysis Dashboard | Power BI

## Description

Developed an interactive Power BI dashboard to analyze Zomato sales performance, customer behavior, restaurant performance, and delivery metrics. The project demonstrates end-to-end business intelligence capabilities, including data modeling, data transformation, DAX calculations, KPI development, and dashboard visualization.

### Key Features

* Total Sales, Total Orders, Average Order Value, and Delivery Performance KPIs
* Sales trend analysis by month and city
* Food category and cuisine-wise revenue analysis
* Top-performing restaurants and customer insights
* Repeat customer analysis and demographic segmentation
* Delivery time performance tracking
* Interactive slicers and drill-through functionality
* Star Schema data modeling for optimized performance
* Advanced DAX measures and calculations
* Row-Level Security (RLS) implementation concepts
* Performance optimization using Power Query and DAX best practices

### Tools & Technologies

* Power BI Desktop
* Power Query
* DAX
* SQL
* Data Modeling (Star Schema)
* Data Visualization

### Business Impact

Enabled data-driven decision-making by providing actionable insights into sales trends, customer preferences, restaurant performance, and operational efficiency through an intuitive and interactive dashboard.
